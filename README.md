## Hera
[[开发文档]](https://www.yuque.com/tal-tech/hera)

### 简介
----
Hera为一个服务孵化器，旨在快速创建适合自己的服务框架，Hera来帮助开发者解决服务本身需要考虑的服务治理、监控、插件等等，开发者可以自己的风格打造属于自己的专属服务框架，在开发过程中大大减少了开发人员在创建服务时的通用配置和初始化工作，让开发者从重复的泥潭里解脱出来！

Hera目前拥有如下孵化能力:

* 孵化Web框架服务

* 孵化Kafka消费服务

* 孵化RPC服务

* 孵化Job服务

同时Hera为服务提供了灵活的hook
* 服务启动hook
